# OW_Wheelpicker

Overwatch 2 Lineup-/Hero-/Map-Picker mit drei Rollenrädern, Hero-Ban-Modus, Open-Queue-Option und OCR-Import.

## Start

```bash
cd owpicker_mvc
python3 main.py
```

## Tests

```bash
python3 -m unittest discover owpicker_mvc/tests
```

## Repo-/Build-Hygiene

```bash
python3 owpicker_mvc/scripts/repo_hygiene.py --include-build
```
